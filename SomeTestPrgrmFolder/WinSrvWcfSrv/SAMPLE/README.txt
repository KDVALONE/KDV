WcfServiceClient - Клиент для WcfServiceEcho, хоститься в собственной службе Windows. (делает следующее, мониторит папку, если появляется файл шлет сообщение на WcfServiceEcho, получает ответ, пишет его в лог )
ConsoleWinClient - Консольный клинет, просто для теста. тоже работате.
